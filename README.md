# Personal Library React Application

A web application that allows users to manage their personal library of books.

## Features

- User registration and login
- CRUD operations for managing books
- User dashboard with a summary of their library
- Responsive UI built with React

## Prerequisites

- Node.js
- NPM (Node Package Manager)
- Heroku CLI (for deployment)

## Installation

1. **Clone the repository**:

    ```bash
    git clone https://github.com/Kelum23/personal-library-ui.git
    cd personal-library-ui
    ```

2. **Install the dependencies**:

    ```bash
    npm install
    ```

3. **Run the application locally**:

    ```bash
    npm start
    ```

    Visit `http://localhost:3000` in your browser.
